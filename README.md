Movie Database – Console Application 

Описание:
Тази програма представлява проста конзолна база данни за филми, написана на езика C. Потребителят може да добавя, разглежда, търси и сортира филми.

Функционалности:
- Добавяне на нов филм
- Показване на всички филми
- Търсене на филм по заглавие
- Сортиране по продължителност
- Сортиране по рейтинг
- Запис и зареждане на данни от файл

Компилация:
gcc movies.c -o movies

Стартиране:
./movies
